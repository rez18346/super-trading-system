# Super Trading System 🚀

Автономная крипто-трейдинг система с ML-ансамблем, real-time WebSocket, SQLite БД и DecisionEngine.

## Возможности
- ML-ансамбль (LSTM, XGBoost, LightGBM) с мультитаймфреймовым анализом
- DecisionEngine — единый центр принятия решений
- Трейлинг-стоп с адаптивным шагом
- Полная синхронизация с биржей
- Дашборд реального времени
- Paper trading / реальная торговля

## Быстрый старт
```bash
pip install -r requirements.txt
python3 main.py
```

Дашборд: http://localhost:8765/
