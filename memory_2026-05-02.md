### Исправлены ошибки

**Проблема 1: MLAdvisor падал с `list indices must be integers, not str`**
- **Причина:** MLAdvisor.evaluate() ожидал pandas DataFrame, а получал список словарей `[{o,h,l,c,v},...]`
- **Исправление:** В decision_engine.py добавлена конвертация `candles_5m` в pandas DataFrame перед передачей в MLAdvisor

**Проблема 2: MTF голос выдавал `error(4)`**
- **Причина:** `_calc_trend_from_candles` ожидал список списков `[ts,o,h,l,c,v]`, а получал список словарей `{o,h,l,c,v,t}`. Словарь не поддерживает индекс `c[4]`
- **Исправление:** В `_calc_trend_from_candles` добавлено определение формата: для словарей берётся `c['c']`, для списков `c[4]`

**Проблема 3: LiquidityCluster падал с `error('c')`**
- **Причина:** 5M свечи передаются как DataFrame → `df.to_dict('records')` с колонками `['close','high','low','volume']`, а модуль ожидал `['c','h','l','v']`
- **Исправление:** `_extract_closes/highs/lows/volumes` теперь поддерживают оба формата: `c/h/l/v` и `close/high/low/volume`
